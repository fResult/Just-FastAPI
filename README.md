# Just Fastapi
FastAPI Learning from the official documentation.
https://fastapi.tiangolo.com/tutorial

## Prerequisite
- [Python 3.10+](https://www.python.org/downloads)
- [Rye](https://rye-up.com/) package manager

## Available Scripts
**Install packages:**
```bash
rye sync
```

**Run:**
```bash
rye run hypercorn src.main:app --reload
```
